# 贡献指南

感谢您对 TradeNote 项目的关注！本文档概述了参与本项目贡献的指南与法律条款。

***

## 1. 如何贡献

我们欢迎以下形式的贡献：

- **内容贡献**：对 Markdown 文档的勘误、补充或改进（如 `TradeNote.md` 及其他 `.md` 文件）
- **代码贡献**：对 `code/` 目录下 Python 代码的 Bug 修复、新脚本或改进
- **问题报告**：提交记录详尽的问题，包括错误、不准确之处或建议
- **Pull 请求**：通过 Pull Request 提交变更以供审核与合并

### 贡献流程

1. Fork 本仓库
2. 创建功能分支（`git checkout -b feature/your-contribution`）
3. 进行修改并使用清晰、描述性的提交信息进行提交
4. 提交 Pull Request，说明变更的目的与范围
5. 等待审核并处理反馈意见

***

本项目采用**双许可模式**。提交贡献即表示您同意以下条款，具体适用取决于您贡献的内容类型：

向本仓库的 Markdown 文件提交任何内容——包括但不限于 `TradeNote.md`、`README.md` 及任何现有或未来的 `.md` 文件——即表示您，作为贡献者，同意将您的贡献按照 [Creative Commons CC0 1.0 Universal 公共领域贡献声明](https://creativecommons.org/publicdomain/zero/1.0/) 献入**公共领域**。

- **放弃**对所贡献内容享有的所有版权及相关权利，在适用法律允许的最大范围内予以放弃；
- **让渡**对所贡献 Markdown 内容的一切所有权、署名权及控制权主张；

向本仓库提交贡献即表示您声明并保证：

1. **原创性**：您的贡献为您本人原创，或您拥有充分权利以适用许可（代码适用 AGPL-3.0，Markdown 内容适用 CC0）提交该贡献
2. **不侵权**：您的贡献不侵犯任何第三方的知识产权、隐私权或其他任何权利
3. **无保密信息**：您的贡献不包含属于任何第三方的保密或专有信息
4. **授权能力**：您具有授予本文档所述权利的法律行为能力和权限
5. **知悉条款**：您已阅读并理解适用于您贡献的许可条款

## 4. CC0 声明的适用范围

CC0 声明**仅**适用于贡献至 Markdown（`.md`）文件的文本内容，**不**适用于：

- 源代码文件（`.py`、`.txt` 及 `code/` 目录下的其他代码文件），此类文件继续适用 AGPL-3.0
- 与本项目相关的任何商标、标识或品牌标识
- 文档中引用或链接的第三方内容，该等内容可能受单独的许可条款约束
