## 🔢 多因子模型

因子分类：价值（PE/PB）、成长（营收增速）、动量（过去N月涨幅）、质量（ROE）、波动率（低波因子）。

🔬 **因子检验**：IC（信息系数）= 因子值与未来收益的秩相关系数，IC均值>0.03且IR（IC均值/IC标准差）>0.5为有效因子。

🧪 **因子合成**：等权法、IC加权法、最优化加权法。

## 🔄 动量策略与均值回归策略

**动量策略**：买入过去N个月涨幅最大的股票，卖出涨幅最小的。时间序列动量看自身趋势，横截面动量看相对排名。

**均值回归策略**：

```python
from statsmodels.tsa.stattools import coint
score, pvalue, _ = coint(stock_a, stock_b)
if pvalue < 0.05:
    spread = stock_a - hedge_ratio * stock_b
    z_score = (spread - spread.mean()) / spread.std()
    if z_score > 2:
        sell_a_buy_b()
    elif z_score < -2:
        buy_a_sell_b()
```

配对交易（Pairs Trading）：找到两个协整的股票，价差偏离均值时做空高估、做多低估。

## 📰 事件驱动策略

基于特定事件（业绩预告超预期、高管增持、分红送转）触发交易。关键是事件数据的及时获取和影响量化。

## ⏰ 择时模型

- **趋势跟踪**：均线金叉/死叉、通道突破（Donchian Channel）
- **市场状态识别**：隐马尔可夫模型（HMM）将市场分为"牛市/熊市/震荡"三种状态

```python
from hmmlearn.hmm import GaussianHMM
model = GaussianHMM(n_components=3, covariance_type='full',
                     n_iter=500, tol=1e-6, random_state=42)
model.fit(returns.values.reshape(-1, 1))
hidden_states = model.predict(returns.values.reshape(-1, 1))
```

## 🧮 衍生品量化

**期权数据获取**：通过券商API或第三方数据源获取期权链数据。

**波动率曲面建模**：用局部波动率模型或SABR模型拟合IV曲面。

**希腊字母对冲自动化**：设定Delta中性目标，当Delta偏离超过阈值时自动对冲。

**轮证量化分析**：筛选IV最低、杠杆适中、街货量合理的轮证，构建评分模型。
