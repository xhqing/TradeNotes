# Lark用户使用说明

Lark用户可以将本项目的 Markdown 文档批量导入为Lark在线文档，方便在Lark中阅读和编辑。

## 导入步骤

1. **新建文件夹** — 在Lark云文档的云盘中新建一个文件夹（例如命名为"交易笔记"）
2. **打开文件夹** — 进入刚创建的文件夹
3. **上传** — 将光标滑动到上方的"上传"按钮处，会自动弹出下拉菜单
4. **选择导入** — 在下拉菜单中选择"导入为在线文档"
5. **批量导入** — 在弹出的界面右上角点击"批量导入"
6. **选择文件** — 在文件系统中找到并选中所有的交易笔记 MD 文档
7. **确认导入** — 点击 Open，Lark会自动将所有 MD 文档批量转化为在线文档

导入完成后，即可在Lark中直接阅读和编辑这些交易笔记。
