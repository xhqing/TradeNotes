## ↕️ 垂直价差策略

垂直价差由同到期日、不同行权价的两腿期权组成，风险和收益均有限。

| 策略               | 构建                  | 方向    | 最大收益       | 最大亏损       |
| :--------------- | :------------------ | :---- | :--------- | :--------- |
| Bull Call Spread | 买低K Call + 卖高K Call | 看涨    | 高K-低K-净权利金 | 净权利金       |
| Bear Put Spread  | 买高K Put + 卖低K Put   | 看跌    | 高K-低K-净权利金 | 净权利金       |
| Bull Put Spread  | 卖高K Put + 买低K Put   | 看涨/中性 | 净权利金       | 高K-低K-净权利金 |
| Bear Call Spread | 卖低K Call + 买高K Call | 看跌/中性 | 净权利金       | 高K-低K-净权利金 |

Bull Call/Bear Put是借方价差（净支出权利金），Bull Put/Bear Call是贷方价差（净收入权利金）。

## 📅 日历价差与对角价差

**日历价差**：卖出近月期权 + 买入远月同行权价期权。利用近月时间衰减快于远月的特性获利。在标的价格接近行权价时收益最大。

**对角价差**：不同行权价 + 不同到期日的组合，融合垂直价差与日历价差的特征。

**财报博弈**：财报前近月IV飙升，远月IV变化较小，日历价差可收割近月IV Crush。

## ⚖️ 比率价差与回填价差

**比率价差**：买入1张期权 + 卖出2张或更多同行权价或不同行权价期权。例如买入1张ATM Call + 卖出2张OTM Call。收益在卖出行权价处最大，但上方风险无限。

**回填价差（Backspread）**：卖出1张低行权价 + 买入2张高行权价（Call Backspread）。大涨时收益无限，大跌时亏损有限。适用于预期大幅波动且方向偏上。

## 🦋 蝶式价差与铁蝶策略

**买入蝶式**：买1张低K + 卖2张中K + 买1张高K（全部Call或全部Put）。收益在标的价格=中K时最大，适合预期标的价格在特定区间。

**铁蝶**：卖1张ATM Put + 卖1张ATM Call + 买1张OTM Put + 买1张OTM Call。本质是卖出跨式+买入保护，收益有限但风险也有限。

## 🔀 跨式与宽跨式

| 策略             | 构建                   | 适用场景         | 最大收益 | 最大亏损 |
| :------------- | :------------------- | :----------- | :--- | :--- |
| Long Straddle  | 买ATM Call + 买ATM Put | 预期大幅波动，方向不确定 | 理论无限 | 总权利金 |
| Short Straddle | 卖ATM Call + 卖ATM Put | 预期横盘         | 总权利金 | 理论无限 |
| Long Strangle  | 买OTM Call + 买OTM Put | 预期大幅波动，成本更低  | 理论无限 | 总权利金 |
| Short Strangle | 卖OTM Call + 卖OTM Put | 预期横盘，收益区间更宽  | 总权利金 | 理论无限 |

🧭 **选择决策**：预期波动大→买跨式/宽跨式；预期波动小→卖跨式/宽跨式。宽跨式成本更低但需要更大波动才能盈利。

## 🦅 铁鹰策略（Iron Condor）

铁鹰是最流行的中性卖方策略之一：

**构建**：卖1张OTM Put + 买1张更OTM Put + 卖1张OTM Call + 买1张更OTM Call

```
收益区间: [Put卖出行权价, Call卖出行权价]
         ↗ 最大收益 = 净权利金
         ↘ 最大亏损 = 行权价间距 - 净权利金
```

🔤 **Greeks剖析**：Delta≈0（中性），Theta为正（每天赚钱），Vega为负（波动率下降赚钱），Gamma为负（标的大幅波动不利）。

🔧 **调整技巧**：

- 标的接近卖出行权价时，可滚动该侧行权价更远
- 加对冲腿（如买入短期ATM期权）限制尾部风险
- 提前平仓：达到最大收益的50-70%时平仓，不必等到期

🪤 **高胜率陷阱**：铁鹰胜率可达70-80%，但一次极端行情的亏损可能抵消数月收益。尾部风险管理是铁鹰的生命线。

## 🦅 秃鹰与定制化组合

秃鹰（Condor）与铁鹰类似，但四腿全部使用同类型期权（全Call或全Put）。定制化组合根据特定市场观点构建，如"看涨但上方有限"可用Bull Call Spread表达。

## 🚪 出场时机与动态调整

- **达到目标收益的50-70%时平仓**：剩余时间价值少，风险收益比恶化
- **止损线**：亏损达到最大收益的1-2倍时平仓
- **调整而非死守**：市场观点变化时，调整行权价或到期日比等待更优

## 🗺️ 不同市场环境下的策略选择矩阵

| 市场环境 | 推荐策略                   | 核心逻辑         |
| :--- | :--------------------- | :----------- |
| 趋势市  | Bull/Bear价差、Backspread | 方向性表达，限制风险   |
| 震荡市  | 铁鹰、铁蝶、Short Strangle   | 收割时间价值与波动率下降 |
| 高波动率 | 日历价差、比率价差              | IV高时卖方有利     |
| 低波动率 | Long Straddle/Strangle | 便宜买波动率       |
| 财报季  | 日历价差、铁蝶                | 收割IV Crush   |
