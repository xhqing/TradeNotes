# 🔴 美股期权卖方策略

## 💡 卖方核心逻辑

卖方的优势在于**概率**：OTM期权到期作废的概率通常高于行权概率。卖方收割时间价值，每过一天Theta都是正收益。

但卖方 ≠ 无风险：卖方的收益有限（权利金），亏损可能远超权利金。卖方是"赚小钱、偶尔亏大钱"的策略，需要严格风控。

## 💰 Cash-Secured Put 卖出策略

💡 **核心逻辑**：卖出1张Put，同时预留足够资金（行权价×100）以备行权时买入正股。

🎯 **适用场景**：

- 想以低于当前市价买入正股
- 在横盘或小幅上涨时赚取权利金收入

🎯 **行权价选择**：以Delta为指引，Delta≈-0.2至-0.3的Put约70-80%概率到期作废，兼顾收益与安全。

📅 **到期日选择**：30-45天，时间衰减效率最高。

📌 **被行权后**：以行权价买入100股正股——这恰好是轮子策略的第一步。

## 📞 Covered Call 卖出策略

💡 **核心逻辑**：持有100股正股 + 卖出1张Call，用正股担保Call的义务。

🎯 **行权价选择**：OTM 1-2档，在"保留上涨空间"与"收取权利金"之间平衡。被行权意味着以行权价卖出正股——如果行权价高于你的目标卖出价，被行权是好事。

💸 **税务考量**（美股）：被行权按行权价计算盈亏，需考虑持有期是否满足长期资本利得条件。

📉 **震荡市价值**：正股不涨不跌时，Covered Call的权利金是唯一的收益来源。

## ⚡ Naked Put / Naked Call

**Naked Put**：卖出Put但不预留全额资金（仅用保证金）。风险与Cash-Secured Put相同，但杠杆更高。尾部风险：标的大跌时亏损 = (行权价 - 标的价格) × 100 - 权利金。

**Naked Call**：卖出Call但不持有正股。**风险无限**——标的理论上可以无限上涨。仅在极少数场景使用（如卖出远OTM Call收取微薄权利金），必须设严格止损。

## 🎡 轮子策略（The Wheel）深度探析

轮子策略是卖方策略的集大成者，循环执行三个步骤：

```
步骤1: 卖出OTM Put → 收取权利金
         ↓ (被行权)
步骤2: 以行权价持有100股正股
         ↓
步骤3: 卖出OTM Covered Call → 收取权利金
         ↓ (被行权，正股被卖出)
回到步骤1，循环往复
```

📋 **选股标准**：

- 你愿意长期持有的优质公司
- 流动性好、期权链活跃
- IV适中偏高（权利金更丰厚）
- 基本面稳健，不会暴跌

🎯 **行权价动态优化**：

- Sell Put：选择你愿意买入的行权价，通常OTM 5-10%
- Sell Covered Call：选择你愿意卖出的行权价，通常OTM 5-10%
- 根据市场趋势调整：牛市中Call行权价选更高，熊市中Put行权价选更低

🔄 **滚动操作**：如果期权快到期但仍ITM，可以"滚动"——平仓当前仓位，开更远到期的新仓位，延长时间以避免行权或等待回归。

📊 **实战回测要点**：

- 牛市：轮子策略收益稳健但可能跑输纯持有
- 震荡市：轮子策略的黄金期，持续收取双份权利金
- 熊市：被行权后正股浮亏，Covered Call权利金提供缓冲

🚀 **进阶变体**：

- 半轮子：只做Sell Put端，不被行权就继续
- 加速轮子：在正股持有期间同时卖出Put（需足够保证金）
- LEAPS轮子：用ITM LEAPS Call替代正股

## 💳 卖方保证金精细化管理

- **组合保证金（Portfolio Margin）**：账户净值>10万美元可申请，按组合风险计算保证金，通常比规则保证金低30-50%
- **保证金使用率**：建议不超过总净值的50%，留足缓冲应对极端行情
- **跨品种对冲**：相关标的的卖方仓位可降低整体保证金需求

## 🧘 卖方策略的交易心理与纪律

- 接受"慢赚"：卖方是概率生意，单笔收益小但胜率高
- 严守止损：标的方向性反转时果断平仓，不要"等回来"
- 控制仓位：单一标的卖方仓位不超过净值的5-10%
- 避免在重大事件前裸卖：财报、FOMC等事件前IV飙升但风险不可控
