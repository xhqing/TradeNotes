# 🧠 交易心理与资金管理

## 🎭 交易心理学核心认知偏误

| 偏误   | 表现              | 应对            |
| :--- | :-------------- | :------------ |
| 损失厌恶 | 亏损的痛苦是等额收益快乐的2倍 | 设定规则化止损，不主观判断 |
| 锚定效应 | 被"买入价"锚定，不愿止损   | 关注当前价值，忽略沉没成本 |
| 确认偏误 | 只关注支持自己观点的信息    | 主动寻找反面证据      |
| 处置效应 | 过早止盈，过晚止损       | 预设止盈止损规则      |
| 过度自信 | 高估自己的判断能力       | 用数据验证，记录预测准确率 |
| 从众心理 | 跟随大众追涨杀跌        | 独立思考，逆向思维     |

## 😰 衍生品交易的特殊心理挑战

- 🎢 **杠杆放大情绪**：5倍杠杆意味着5倍的情绪波动，恐惧和贪婪都被放大
- ⏰ **时间损耗焦虑**：买方看着权利金每天减少，心理压力巨大
- 💥 **被行权的心理冲击**：卖Put被行权后"被迫"持有正股，可能产生抵触情绪
- 💀 **被收回的冲击**：牛熊证被强制收回，资金瞬间大幅缩水

💪 **心理建设**：接受衍生品交易的特殊性，将"被行权"视为"以目标价买入"而非亏损，将"时间损耗"视为"保险费"或"租金"。

## 📏 建立交易纪律的方法论

1. **交易计划书面化**：每笔交易前写下逻辑、仓位、止损
2. **执行与计划分离**：计划时理性，执行时机械
3. **定期复盘**：每周/月回顾交易日志，识别模式
4. **情绪记录**：记录每笔交易时的情绪状态，识别情绪对决策的影响

## 📐 资金管理的数学基础

承接凯利公式，资金管理的核心问题是：**在给定的风险约束下，如何分配资金以最大化长期增长？**

凯利公式提供了理论最优解，但实际应用需要考虑：

- 参数估计的不确定性
- 收益分布的非正态性
- 流动性和交易成本约束

## ⚖️ 固定比例 vs 固定金额风险模型

**固定比例法**：每笔交易风险不超过净值的1-2%。仓位 = (净值 × 风险比例) / 止损距离。

**固定金额法**：每笔交易投入固定金额。简单但无法随账户增长调整。

实践中，固定比例法更优——它天然实现了"赢时加仓、输时减仓"的效果。

## 📉 最大回撤控制体系

- **回撤熔断**：当回撤达到X%（如15%）时，暂停交易，冷静期至少1周
- **分阶段恢复**：恢复交易后仓位减半，逐步恢复到正常水平
- **年度回撤上限**：设定年度最大回撤（如25%），触及则全年停止交易

## 🔗 相关性风险与组合管理

不同衍生品之间可能存在隐蔽相关性：

- 同一标的的卖Put和卖Call，在暴跌时同时亏损（相关性趋1）
- 跨市场相关性在危机时急剧上升

**应对**：计算组合的条件相关性（危机时的相关性），而非无条件相关性。

## 🦢 黑天鹅事件应对策略

- 🛡️ **尾部对冲**：持续买入远OTM Put作为保险，成本约年化1-3%
- 🎬 **情景分析**：预设极端情景（如标普单日跌10%），计算组合最大亏损
- 💪 **压力测试**：基于历史极端事件（2008金融危机、2020疫情）测试组合表现

## 🛡️ 衍生品交易特殊风险控制

- **保证金监控**：保证金使用率>60%时预警，>80%时强制减仓
- **到期日管理**：避免大量期权同一天到期，分散到期日
- **隔夜风险**：重大事件前减仓或对冲，避免裸卖方过夜

## ⛓️ 杠杆使用的铁律

杠杆的数学本质：收益和亏损等比例放大。10倍杠杆下，10%的下跌意味着100%的亏损。

**铁律**：

1. 总杠杆不超过凯利公式的半凯利建议
2. 单一标的杠杆不超过总净值的20%
3. 永远预留至少30%的现金缓冲
4. 杠杆是工具不是目标——只有在明确优势时才使用
