## 💡 凯利公式的起源与直觉理解

1956年，贝尔实验室的约翰·凯利发表论文《信息率的新解释》，将香农的信息论与赌博下注联系起来。核心洞察：**在有利可图的赌局中，下注比例决定了长期财富增长率**。

📈 **几何增长 vs 算术增长**：假设你每次下注全部资金，赢翻倍输归零。即使胜率60%，长期必归零——因为一次失败就清零。但如果每次只下注一部分，长期财富以几何增长率复合增长。

🧮 **为什么最大化对数期望**：最大化E\[log(W)]等价于最大化长期复合增长率。这是唯一使得"几乎必然不破产"且"长期财富最大"的目标函数。

## 📐 经典凯利公式数学推导

考虑一个二元赌局：胜率p，净赔率b（下注1元，赢得b元，输则损失1元），败率q = 1-p。

设下注比例为f，则一次赌局后财富的倍数为：

- 赢：W × (1 + fb)
- 输：W × (1 - f)

最大化期望对数财富：

$$G(f) = p \cdot \ln(1 + fb) + q \cdot \ln(1 - f)$$

对f求导并令其为零：

$$\frac{dG}{df} = \frac{pb}{1+fb} - \frac{q}{1-f} = 0$$

解得：

$$\boxed{f^\* = \frac{bp - q}{b} = p - \frac{q}{b}}$$

💡 **直观理解**：

- 分子bp - q是"边际优势"——期望净收益率
- 分母b是"单位风险收益"
- f\*就是"优势除以赔率"

📌 **前提假设**：

1. 赌局可无限重复
2. 每次独立同分布
3. 可以下注任意比例
4. 目标是最大化长期复合增长率

## 🔗 连续凯利公式与投资映射

当收益服从正态分布N(μ, σ²)时，凯利比例简化为：

$$\boxed{f^\* = \frac{\mu}{\sigma^2}}$$

其中μ是期望超额收益，σ²是收益方差。

🔗 **夏普比率与凯利的联系**：夏普比率 SR = μ/σ，代入得 f\* = SR/σ。这意味着：

- 相同夏普比率的策略，波动率越小的越值得加大仓位
- 凯利公式本质上是用夏普比率除以波动率来决定仓位

🌐 **广义凯利**：对于任意收益分布，最优f*满足 E\[r/(1+f*r)] = 0，其中r是收益率。这需要数值求解。

## ✂️ 分数凯利策略

全凯利（100% Kelly）在实践中波动极大。模拟显示：

| 策略     | 长期增长率  | 年化波动率 | 最大回撤（典型） |
| :----- | :----- | :---- | :------- |
| 全凯利    | 最高     | 极高    | 50-90%   |
| 半凯利    | 约75%最高 | 约50%  | 20-40%   |
| 四分之一凯利 | 约50%最高 | 约25%  | 10-20%   |

半凯利（Half-Kelly）是最流行的折中方案：牺牲25%的增长率，换取波动率和回撤大幅降低。

❓ **"几分之几凯利最适合我？"** 取决于你的风险承受能力和对回撤的容忍度。经验法则：

- 能承受50%+回撤：全凯利
- 能承受30%回撤：半凯利
- 只能承受15%回撤：四分之一凯利

## 📊 多资产多元凯利

当有N个资产时，凯利公式推广为矩阵形式：

$$\boxed{\mathbf{f^\*} = \Sigma^{-1} \cdot \boldsymbol{\mu}}$$

其中Σ是收益协方差矩阵，μ是期望超额收益向量。

📐 **与均值方差优化的关系**：凯利最优组合等价于最大化夏普比率的无约束切线组合。如果加入杠杆约束（如f之和≤1），则退化为约束优化问题。

⛓️ **杠杆约束下的凯利**：

$$\max\_{\mathbf{f}} \boldsymbol{\mu}^T \mathbf{f} - \frac{1}{2} \mathbf{f}^T \Sigma \mathbf{f} \quad \text{s.t.} \quad \mathbf{1}^T \mathbf{f} \leq L\_{max}$$

## 📈 凯利公式在股票交易中的应用

**从历史记录逆向估算**：

```python
def estimate_kelly_params(trades):
    wins = trades[trades > 0]
    losses = trades[trades < 0]
    p = len(wins) / len(trades)
    b = wins.mean() / abs(losses.mean())
    f_star = (b * p - (1 - p)) / b
    return p, b, f_star
```

**趋势跟踪中的凯利**：趋势策略胜率通常30-40%，但盈亏比高（2:1到5:1），凯利比例通常15-25%。

**均值回归中的凯利**：胜率较高（55-65%），盈亏比适中（1:1到1.5:1），凯利比例通常10-20%。

**A股涨跌停约束**：涨跌停板限制了单日最大亏损，相当于降低了σ，理论上凯利比例可以更高，但流动性风险（无法止损）需要额外折价。

## 🔥 凯利公式在衍生品交易中的应用（重点）

### 🔴 期权卖方策略

🎯 **"胜率"定义**：卖出OTM Put的胜率≈1-Delta。例如Delta=-0.2的Put，约80%概率到期作废。但胜率高≠期望收益为正——20%的亏损可能远超80%的收益。

💥 **尾部风险冲击**：卖方策略的收益分布是"赚小钱、偶尔亏大钱"，严重偏离正态。一次极端亏损可能抵消数月收益。全凯利在这种分布下严重高估仓位。

🔧 **条件凯利修正**：

$$f^\*\_{adjusted} = \frac{\mu}{\sigma^2} \times \frac{1}{1 + \text{TailRiskMultiplier}}$$

其中TailRiskMultiplier根据历史最大亏损/平均收益的比率计算，通常取2-5倍。

### 🟢 期权买方策略

买方是"低胜率高赔率"策略。凯利公式天然适配这种场景：

```python
p = 0.30
b = 3.0
f_star = (b * p - (1 - p)) / b
print(f"凯利比例: {f_star:.2%}")
```

输出：凯利比例 = 6.67%。买方仓位天然应该很小。

### 🎡 轮子策略的凯利头寸优化

轮子策略同时涉及卖Put和卖Call，需要综合计算：

1. 估算Sell Put阶段的胜率和盈亏比
2. 估算持有正股+Sell Call阶段的胜率和盈亏比
3. 加权平均得到轮子策略的整体参数
4. 代入凯利公式，再取半凯利

实践中，轮子策略的半凯利通常建议单标的仓位不超过净值的10-15%。

### 🇭🇰 轮证/牛熊证的凯利应用

**收回风险折价**：牛熊证有强制收回风险，需要在凯利公式中额外扣除收回概率对应的损失。

$$f^*\_{CBBC} = f^*_{standard} \times (1 - P_{knockout})$$

**时间损耗调整**：窝轮的时间损耗是确定性成本，降低了期望收益μ，从而降低凯利比例。

### 🔀 跨策略凯利

同时管理多组衍生品头寸时，使用多元凯利公式，将所有头寸的收益和协方差纳入矩阵计算。

## ⚠️ 凯利公式的局限性与改进

⚠️ **参数估计误差放大**：凯利公式对输入参数极其敏感。胜率偏差1%可能导致仓位偏差20%以上。

🛡️ **稳健凯利**：使用参数的置信区间下界而非点估计：

$$f^\*\_{robust} = \frac{\hat{\mu} - c \cdot SE(\hat{\mu})}{\hat{\sigma}^2}$$

其中c是保守系数（如1.0-2.0）。

🐉 **非正态厚尾修正**：金融收益分布的峰度远高于正态分布。加入高阶矩修正：

$$f^\*\_{adjusted} = \frac{\mu}{\sigma^2} - \frac{\mu \cdot \kappa}{2\sigma^4}$$

其中κ是超额峰度。厚尾分布下κ>0，修正后的凯利比例更低。

🔄 **动态凯利与贝叶斯更新**：

```python
class BayesianKelly:
    def __init__(self, alpha=1, beta=1):
        self.alpha = alpha
        self.beta = beta

    def update(self, win):
        if win:
            self.alpha += 1
        else:
            self.beta += 1

    def kelly(self, b):
        p = self.alpha / (self.alpha + self.beta)
        return (b * p - (1 - p)) / b
```

每笔交易后更新胜率估计，凯利比例动态调整。初期参数不确定时仓位小，随着数据积累逐渐增大。

## 🔍 凯利 vs 其他头寸管理方法全景对比

| 方法       | 仓位计算                 | 优点        | 缺点        |
| :------- | :------------------- | :-------- | :-------- |
| 凯利公式     | f\* = (bp-q)/b       | 理论最优长期增长  | 对参数敏感，波动大 |
| 固定比例法    | 每次投入净值X%             | 简单易执行     | 不考虑策略优势   |
| 固定金额法    | 每次投入固定金额             | 最简单       | 不随账户增长调整  |
| 风险平价     | σ⁻¹加权                | 风险均衡      | 忽略收益差异    |
| 凯利+VaR约束 | min(f\*, VaR\_limit) | 兼顾增长与风险控制 | 约束可能过紧    |

**凯利与VaR/CVaR结合**：在凯利最优仓位基础上，加入"最大单日亏损不超过X%"的约束，通过CVaR优化求解。

## 💻 Python实现凯利公式

### 1️⃣ 单资产经典凯利

```python
def kelly_classic(p, b):
    q = 1 - p
    f = (b * p - q) / b
    return max(f, 0)

print(kelly_classic(0.6, 1.0))
```

### 🌐 多元凯利

```python
import numpy as np

def kelly_multi(mu, sigma):
    return np.linalg.solve(sigma, mu)

mu = np.array([0.10, 0.08])
sigma = np.array([[0.04, 0.01], [0.01, 0.03]])
f = kelly_multi(mu, sigma)
print(f"凯利配置: {f}")
```

### 🎲 蒙特卡洛模拟

```python
def simulate_wealth(f, p, b, n_rounds=1000, n_paths=1000):
    final_wealth = []
    for _ in range(n_paths):
        wealth = 1.0
        for _ in range(n_rounds):
            if np.random.random() < p:
                wealth *= (1 + f * b)
            else:
                wealth *= (1 - f)
        final_wealth.append(wealth)
    return np.median(final_wealth)

for f_label, f in [('全凯利', 0.2), ('半凯利', 0.1), ('2倍凯利', 0.4)]:
    median = simulate_wealth(f, 0.6, 1.0)
    print(f"{f_label}(f={f}): 中位终值 = {median:.2f}")
```

### 🔄 动态凯利（贝叶斯更新）

```python
class DynamicKelly:
    def __init__(self, prior_alpha=10, prior_beta=10):
        self.alpha = prior_alpha
        self.beta = prior_beta

    def update(self, win):
        self.alpha += int(win)
        self.beta += int(not win)

    def get_f(self, b):
        p = self.alpha / (self.alpha + self.beta)
        return max((b * p - (1 - p)) / b, 0)

dk = DynamicKelly()
for result in [1,1,0,1,0,1,1,0,1,1]:
    dk.update(result)
    print(f"胜率估计: {dk.alpha/(dk.alpha+dk.beta):.2f}, 凯利: {dk.get_f(1.5):.2%}")
```

## 📋 凯利公式实战案例深度剖析

### 案例一：卖出SPY OTM Put的仓位计算

SPY当前价450，卖出30天到期、行权价440的Put，Delta=-0.15。

- 胜率 p ≈ 1 - 0.15 = 0.85
- 权利金收入 = $1.50（每份$150）
- 最大亏损 = (440 - 0) × 100 - 150 = $43,850（极端情况）
- 实际亏损（标跌5%至427.5）= (440-427.5)×100 - 150 = $1,100
- 净赔率 b = 150 / 1100 = 0.136

凯利：f\* = (0.136 × 0.85 - 0.15) / 0.136 = -0.21... 负值！

这说明用极端亏损计算赔率时，凯利建议不下注。需要用"典型亏损"而非极端亏损：

- 典型亏损（标跌2%至441，刚好ITM）= (440-441)×100 - 150 = -$50（实际还赚$100）
- 修正b = 150 / 300 = 0.5（假设平均亏损$300）
- f\* = (0.5 × 0.85 - 0.15) / 0.5 = 0.55

半凯利 = 27.5%。但考虑到尾部风险，实际建议不超过净值的10-15%。

### 案例二：轮子策略中的头寸加码

轮子策略历史胜率70%，平均赢$200/平均亏$500，b=0.4。

f\* = (0.4 × 0.7 - 0.3) / 0.4 = -0.05... 又是负值？

问题在于卖方策略的盈亏比天然<1。需要用连续凯利：

- 月均收益μ = 0.7×200 - 0.3×500 = -$10（期望为负？）
- 实际上轮子策略的收益来自正股增值+权利金，需要综合计算
- 综合月均收益约2-3%，月波动率约5%
- f\* = 0.025 / 0.0025 = 10（连续凯利建议10倍杠杆？）

这显然不合理——连续凯利假设正态分布，但轮子策略有厚尾。取半凯利5倍仍过高。加入尾部风险修正后，建议1-2倍杠杆，即单标的仓位不超过净值的15-20%。

### 案例三：牛熊证的高杠杆凯利边际

恒指牛证，收回价距当前2%，有效杠杆15倍。

- 收回概率（30天）≈ 30%
- 未收回时收益 ≈ 5%（恒指涨2%×15倍杠杆-时间成本）
- 被收回时亏损 ≈ 80-100%

凯利：p=0.70, b=0.05/0.90=0.056

f\* = (0.056 × 0.70 - 0.30) / 0.056 = -4.14

凯利强烈建议不下注！高杠杆牛熊证的期望收益为负，凯利公式揭示了这一点。

### 案例五：凯利从理论到实盘的现实修正

理论凯利→实盘凯利的修正清单：

1. **参数不确定性折价**：使用置信区间下界，折价20-30%
2. **厚尾修正**：根据超额峰度调低，折价10-20%
3. **流动性约束**：大仓位冲击成本，折价5-10%
4. **心理承受力**：如果全凯利回撤让你睡不着，再折价50%
5. **相关性不确定性**：危机时相关性趋1，多元凯利高估分散效果，折价10-15%

最终实盘仓位 ≈ 理论凯利 × 0.3 \~ 0.5

## 🧘 凯利公式的哲学反思

🧘 **数学最优≠心理最优**：全凯利可能导致50%以上的回撤，绝大多数交易者无法承受。半凯利或四分之一凯利在心理上更可持续，而长期增长率只降低25-50%。

🪄 **"永不爆仓"的神话**：凯利公式在理论上保证"几乎必然不破产"，但前提是：

1. 下注比例可以无限细分
2. 没有最小交易单位限制
3. 参数估计完全准确

现实中这三个条件都不成立。凯利公式降低破产概率，但不保证永不爆仓。

📈 **交易者的凯利曲线**：每个交易者都有一条属于自己的凯利曲线——在仓位-增长率平面上，存在一个让你"既睡得着觉，又能持续赚钱"的最优点。找到这个点，比追求理论最优更重要。
