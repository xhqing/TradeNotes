# 🔀 衍生品基础概念

## 📑 什么是金融衍生品

金融衍生品是其价值依赖于标的资产（股票、指数、利率、汇率等）的金融合约。衍生品的本质是**权利与义务的合约**——它本身不创造价值，而是重新分配风险与收益。

📂 **四大分类**：

| 类型   | 定义               | 典型应用    |
| :--- | :--------------- | :------ |
| 远期合约 | 双方约定未来以固定价格交易    | 外汇远期    |
| 期货合约 | 标准化远期，交易所交易      | 股指期货    |
| 期权合约 | 买方有权（非义务）以约定价格交易 | 看涨/看跌期权 |
| 互换合约 | 双方交换现金流          | 利率互换    |

## ⚖️ 定价基础

- **内在价值**：立即行权可获得的收益。看涨期权内在价值 = max(S - K, 0)
- **时间价值**：权利金超出内在价值的部分，反映未来价格波动的可能性
- **无套利定价**：如果两个投资组合在未来所有状态下有相同收益，则当前价格必须相同
- **风险中性定价**：在风险中性世界中，所有资产的期望收益率等于无风险利率

## 🔧 保证金与杠杆机制

保证金是衍生品交易的核心风控工具：

- 💵 **初始保证金**：开仓时必须缴纳的最低资金
- 🔒 **维持保证金**：持仓期间必须维持的最低资金水平
- 🚫 **强制平仓**：保证金低于维持水平时，经纪商强制平仓

杠杆 = 标的资产价值 / 保证金。杠杆放大收益的同时等比例放大亏损。

## 👥 市场参与者生态

- **套期保值者**：利用衍生品对冲现有风险（如航空公司锁定燃油成本）
- **投机者**：主动承担风险以获取利润
- **套利者**：利用价格差异获取无风险收益
- **做市商**：提供买卖报价赚取价差，维持市场流动性
